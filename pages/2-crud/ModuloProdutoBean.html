ModuloProdutoBean novo arquivo

O Bean é a classe de dados que representa um único registro da tabela laboratorio.pro. Ele define os campos, os tipos e a chave primária — é a estrutura que o DAO, o formulário e a listagem usam para ler, gravar e exibir os dados.

package br.xt.app.modulo.produto;

import br.jasap.dao.DBInfo;
import java.io.Serializable;

public class ModuloProdutoBean implements Serializable {

    public static String TABLE = "laboratorio.pro";

    private Integer id_pro;
    private String  nome_pro;
    private Double  vl_pro;
    private Integer qtd_pro;
    private String  obs_pro;

    @DBInfo(serial=true, pk=true)
    public Integer getId_pro() { return id_pro; }
    public void setId_pro(Integer id_pro) { this.id_pro = id_pro; }

    public String getNome_pro() { return nome_pro; }
    public void setNome_pro(String nome_pro) { this.nome_pro = nome_pro; }

    public Double getVl_pro() { return vl_pro; }
    public void setVl_pro(Double vl_pro) { this.vl_pro = vl_pro; }

    public Integer getQtd_pro() { return qtd_pro; }
    public void setQtd_pro(Integer qtd_pro) { this.qtd_pro = qtd_pro; }

    public String getObs_pro() { return obs_pro; }
    public void setObs_pro(String obs_pro) { this.obs_pro = obs_pro; }

    public static String ID_PRO   = "id_pro";
    public static String NOME_PRO = "nome_pro";
    public static String VL_PRO   = "vl_pro";
    public static String QTD_PRO  = "qtd_pro";
    public static String OBS_PRO  = "obs_pro";

}

ElementoO que significa
publicA classe é visível para todo o projeto — o DAO, o formulário e a listagem precisam instanciá-la diretamente
classDeclara que isso é uma classe Java — o bloco que agrupa os dados de um produto
ModuloProdutoBeanNome da classe. Convenção do projeto: prefixo do módulo + entidade + Bean
implements SerializablePermite que o objeto seja armazenado na sessão HTTP. O Jasap guarda o bean na sessão entre requisições — sem Serializable isso causaria um erro em runtime ao tentar persistir o estado

public static String TABLE = "laboratorio.pro"

ElementoO que significa
publicO DAO e outras classes precisam ler essa constante diretamente via ModuloProdutoBean.TABLE
staticPertence à classe, não a uma instância — permite usar ModuloProdutoBean.TABLE sem criar um new ModuloProdutoBean()
StringTipo da constante — é um texto com o nome da tabela

"laboratorio.pro"Schema + tabela no PostgreSQL. O schema laboratorio agrupa as tabelas deste domínio. O DAO passa essa constante para insert(), update() e delete()

private Integer id_pro / private String nome_pro / ...

Cada campo privado representa uma coluna da tabela. O Jasap usa os nomes dos getters (via reflection) para mapear automaticamente os valores do ResultSet para o bean em query.populateBean(bean) — por isso o nome do campo deve bater exatamente com o nome da coluna no banco.

@DBInfo(serial=true, pk=true)

Anotação do Jasap que configura o comportamento do campo no banco de dados. Fica no getter, não no campo.

AtributoO que faz
serial=trueIndica que o banco gera o valor automaticamente no INSERT via sequence. O DAO omite esse campo no SQL de INSERT — se fosse enviado, causaria conflito com o DEFAULT nextval() do PostgreSQL
pk=trueIdentifica este campo como chave primária. O DAO usa esse campo no WHERE do UPDATE e do DELETE — sem isso o framework não saberia qual registro atualizar

public static String ID_PRO = "id_pro"

Constantes que referenciam o nome das colunas no banco. Usadas no DAO ao construir SQLs e no WBean ao montar filtros — evitam strings literais espalhadas pelo código.
