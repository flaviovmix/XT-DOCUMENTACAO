ModuloProdutoBean

Classe de dados que representa um registro da tabela. Define os campos, os tipos e a chave primária.

ModuloProdutoWBean

Bean de filtro da listagem. Implementa Criteria e carrega os parâmetros que o DAO usa para montar o WHERE do SQL.

Ver docs

ModuloProdutoDAO

Camada de acesso ao banco. Único arquivo que fala diretamente com o PostgreSQL — listagem, busca, insert, update e delete.

Ver docs

ModuloProdutoModel

Camada de regras de negócio entre o formulário e o DAO. Neste CRUD simples delega diretamente — mas é aqui que as regras entrariam.

Ver docs

ModuloProdutoAction

Classe base abstrata de List e Form. Centraliza ROOT, proBean(), proWBean() e id_pro() para todas as actions de Produto.

Ver docs

ModuloProdutoList

Tela de listagem. Exibe os registros em uma ListView paginável com ordenação por coluna, botão para novo registro e exclusão inline com confirmação.
