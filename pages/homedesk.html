Monta o menu lateral do sistema (tela inicial após o login). Adicionar a entrada do módulo para que ele apareça no menu principal.

// Adicionar os imports:
import br.xt.app.modulo.ModuloHome;
import br.xt.app.modulo.ModuloManager;

// Adicionar dentro de menuDesk():
m.add(ok(ModuloManager.F_ACESSO_MODULO),
        "MÓDULO",
        link(ModuloHome.class).childWindow("__WD_XT_MODULO"),
        null).setImage("res/ico/mn_cto_clientes.png");

ok(ModuloManager.F_ACESSO_MODULO)

Verifica se o usuário logado tem permissão para acessar o módulo. Se não tiver, o item simplesmente não aparece no menu — sem erro, sem mensagem.

childWindow("__WD_XT_MODULO")

Abre o módulo em uma janela filha dedicada. Se o usuário clicar novamente no item, a janela já existente é reutilizada — não abre duplicada.
