ModuloMdFactory novo

Centraliza o acesso a todos os Models/DAOs do módulo. Qualquer action do sistema pode acessar os dados via getFactory().modulo().xxxModel(), sem instanciar o model diretamente.

package br.xt.app.modulo;

import br.jasap.core.AppManager;

public class ModuloMdFactory {

    protected AppManager manager = null;

    public ModuloMdFactory() {
    }

    public ModuloMdFactory(AppManager manager) {
        this.manager = manager;
    }

    public AppManager getManager() {
        return manager;
    }

}

Como cresce

Conforme entidades forem criadas, os models são adicionados aqui com o padrão lazy init:

private ModuloXxxModel xxxModel = null;
public ModuloXxxModel xxxModel() {
    if (xxxModel == null) xxxModel = new ModuloXxxModel(manager);
    return xxxModel;
}
