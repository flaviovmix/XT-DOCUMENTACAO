AppsRootModelFactory editado

Centraliza o acesso a todas as factories do sistema. Adicionar o método modulo() para expor o ModuloMdFactory.

// Adicionar o import:
import br.xt.app.modulo.ModuloMdFactory;

// Adicionar o método:
public ModuloMdFactory modulo() { return new ModuloMdFactory(getManager()); }

Por que é necessário

Após essa adição, qualquer action do sistema pode acessar os dados do módulo via:

getFactory().modulo().xxxModel()

A instância de ModuloMdFactory é criada com o manager correto, garantindo o acesso ao banco de dados e à sessão.
