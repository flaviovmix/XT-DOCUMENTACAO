RootManager editado

Ponto de inicialização de toda a aplicação. Adicionar a chamada ao ModuloManager para que o framework conheça o módulo.

// Adicionar o import:
import br.xt.app.modulo.ModuloManager;

// Adicionar dentro de config():
new ModuloManager().config();

Por que é necessário

Sem essa linha, o ModuloManager.config() nunca é executado — nenhum regAction() é chamado e o módulo inteiro é invisível para o Jasap. Nenhuma requisição chegaria a ser roteada para suas actions.
